## My key bindings

**M-i** - Up
**M-k** - Down
**M-j** - Left
**M-l** - Right

**M-u** - На слово назад
**M-o** - На слово вперед
**M-a** - В начало строки
**M-e** - В конец строки
**C-a** - В начало функции
**C-e** - В конец функции
**M-n** - Прокрутка вверх
**M-b** - Прокрутка вниз
**M-,** - В начало буфера
**M-.** - В конец буфера
**M-[** - backward-list
**M-]** - forward-list

**M-x** - вырезать
**M-c** - копировать
**M-v** - вставить
**M-d** - удалить символ назад
**M-f** - удалить символ вперед

**C-f** - открыть файл
**M-b** - переключение буферов
**C-d** - удалить буфер
**M-g** - сохранить буфер
**C-w** - записать файл
**C-q** - exit with save
**C-0** - удалить текущее окно
**C-1** - удалить другие окна
**С-2** - разбить окно горизонтально
**C-3** - -- вертикально
**C-S-i** - по окну вверх
**C-S-k** - down
**C-S-j** - left
**C-S-l** - right

**M-z** - отменить
**M-q** - еще не понял что делает
